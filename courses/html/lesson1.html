Welcome to HTML! 👋

Today, we're going to learn how to create your very first web page! HTML is like building blocks for websites - it's the foundation of everything you see on the web.

What is HTML? 🤔

HTML stands for HyperText Markup Language. Think of it as giving instructions to your computer about what to show on a webpage:

  • 🏗️ It's the structure of your webpage
  • 📝 It tells browsers what each piece of content is
  • 🎨 It's like a blueprint for your website

Let's Break It Down! 📚

<!DOCTYPE html>

This tells the browser we're using modern HTML

<html>

The container for all your HTML code

<head>

Contains information about your page

<body>

Where all the visible content goes
