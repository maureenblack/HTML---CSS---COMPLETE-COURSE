HTML elements are the building blocks of web pages. In this lesson, we'll learn about the most common HTML elements and how to use them effectively.

Common HTML Elements üìö

Text Elements

  • <h1> to <h6> - Headings (from most to least important)
  • <p> - Paragraphs
  • <strong> - Bold text
  • <em> - Italic text
  • <br> - Line break
  • <hr> - Horizontal rule

Understanding the Elements

  • Headings (<h1> to <h6>): Used for titles and subtitles. H1 is the most important, H6 is the least important.
  • Paragraphs (<p>): Used for blocks of text.
  • Text Formatting:
    • <strong> makes text bold (for importance)
    • <em> makes text italic (for emphasis)
  • Line Breaks:
    • <br> creates a single line break
    • <hr> creates a horizontal line to separate content
