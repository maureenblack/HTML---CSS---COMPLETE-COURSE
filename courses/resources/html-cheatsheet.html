HTML Cheat Sheet

Document Structure

<!DOCTYPE html>
Declares the document as HTML5
<html>
Root element of the HTML page
<head>
Contains metadata about the document
<body>
Contains the visible page content

Text Content

<h1> to <h6>
Headings (h1 is largest, h6 is smallest)
<h1>Main Title</h1>
<p>
Paragraph of text
<p>Your text here</p>
<strong>
Important text (displayed bold)
<strong>Important!</strong>
<em>
Emphasized text (displayed italic)
<em>Emphasized</em>

Lists

<ul>
Unordered list (bullet points)
<ul> <li>Item 1</li> </ul>
<ol>
Ordered list (numbers)
<ol> <li>First item</li> </ol>
<li>
List item
<li>List item</li>

Links and Images

<a>
Creates a hyperlink
<a href="url">Link text</a>
<img>
Embeds an image
<img src="image.jpg" alt="Description">

Forms

<form>
Creates a form for user input
<form action="/submit"></form>
<input>
Input field (text, password, etc.)
<input type="text" name="username">
<button>
Clickable button
<button type="submit">Send</button>
