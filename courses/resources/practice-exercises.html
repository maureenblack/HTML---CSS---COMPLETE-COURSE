HTML Practice Exercises

Exercise 1: Basic Page Structure

Task: Create a Simple Profile Page

Create an HTML page that includes:

  • A main heading with your name
  • A subheading with "About Me"
  • A paragraph about yourself
  • A list of your hobbies
Hint: Remember to use appropriate heading tags (h1, h2) and list elements (ul, li).
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My Profile</title> </head> <body> <h1>John Doe</h1> <h2>About Me</h2> <p>I am a student learning web development. I love creating websites and learning new technologies.</p> <h2>My Hobbies</h2> <ul> <li>Coding</li> <li>Reading</li> <li>Playing Guitar</li> </ul> </body> </html>

Exercise 2: Text Formatting

Task: Create a Recipe Page

Create a recipe page that includes:

  • A title for your recipe
  • A list of ingredients (using unordered list)
  • Steps to make the recipe (using ordered list)
  • Use text formatting (bold and italic) to emphasize important parts
Hint: Use <strong> for important text and <em> for emphasis.
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My Favorite Recipe</title> </head> <body> <h1>Chocolate Chip Cookies</h1> <h2>Ingredients</h2> <ul> <li><strong>2 cups</strong> flour</li> <li><strong>1 cup</strong> butter</li> <li><strong>1 cup</strong> chocolate chips</li> </ul> <h2>Instructions</h2> <ol> <li>Preheat oven to <em>350Â°F</em></li> <li>Mix ingredients until <em>well combined</em></li> <li>Bake for <strong>10-12 minutes</strong></li> </ol> </body> </html>

Exercise 3: Links and Navigation

Task: Create a Mini Website

Create a small website with:

  • A home page (index.html)
  • An about page
  • A contact page
  • Navigation links between all pages
Hint: Use relative paths in your href attributes to link between pages.
<!-- index.html --> <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Home</title> </head> <body> <nav> <a href="index.html">Home</a> | <a href="about.html">About</a> | <a href="contact.html">Contact</a> </nav> <h1>Welcome to My Website</h1> <p>This is the home page.</p> </body> </html>
