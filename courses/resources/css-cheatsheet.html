CSS Cheat Sheet

Text Styling

color
Sets the color of text
color: blue;
Blue text example
font-size
Sets the size of the font
font-size: 20px;
Larger text example
font-weight
Sets the thickness of text
font-weight: bold;
Bold text example

Box Model

margin
Sets the outer spacing of an element
margin: 10px;
Box with margin
padding
Sets the inner spacing of an element
padding: 10px;
Box with padding
border
Sets the border of an element
border: 2px solid black;
Box with border

Layout

display
Sets how an element should be displayed
display: flex;
position
Sets the positioning method
position: relative;
float
Sets how an element should float
float: left;

Colors & Background

background-color
Sets the background color
background-color: #ff0000;
Red background
background-image
Sets a background image
background-image: url('image.jpg');
opacity
Sets the transparency
opacity: 0.5;
50% opacity
