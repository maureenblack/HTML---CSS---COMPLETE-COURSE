Media Queries üìè

Common Breakpoints

  • Mobile: max-width: 600px
  • Tablet: max-width: 900px
  • Desktop: min-width: 901px

Media Query Syntax

@media (max-width: 600px) { /* Mobile styles */ } @media (min-width: 601px) and (max-width: 900px) { /* Tablet styles */ }

Responsive Units üìê

Relative Units

  • em - relative to parent
  • rem - relative to root
  • vw - viewport width
  • vh - viewport height
  • % - percentage of parent

When to Use

  • Font sizes: rem
  • Padding/Margin: em
  • Container widths: %
  • Full-screen elements: vh/vw

Mobile First Design üì±

Why Mobile First?

  • Most users are on mobile
  • Forces focus on content
  • Easier to scale up than down
  • Better performance

Best Practices

  • Start with mobile layout
  • Use min-width queries
  • Keep layouts simple
  • Test on real devices

Pro Tips! üí°

  • Always include the viewport meta tag
  • Use fluid grids and flexible images
  • Test on multiple devices
  • Consider touch targets on mobile
