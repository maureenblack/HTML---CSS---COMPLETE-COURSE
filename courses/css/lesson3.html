Understanding the Box Model üì¶

Every element in HTML is a box! Let's learn how to control these boxes with CSS.

The Box Model Parts üéÅ

Content

The actual content of the box:

  • width
  • height
  • min-width
  • max-width

Padding

Space inside the border:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Border

The line around the box:

  • border-width
  • border-style
  • border-color
  • border-radius

Margin

Space outside the border:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Display Properties üñ•Ô∏è

Block

Takes full width, starts on new line:

display: block;

Inline

Takes only needed width:

display: inline;

Inline-Block

Mix of both:

display: inline-block;

Box Sizing üìè

Content-Box

Default behavior:

box-sizing: content-box;

Width and height only include content

Border-Box

Modern approach:

box-sizing: border-box;

Width and height include padding and border

Pro Tips! üí°

  • Use border-box for easier sizing calculations
  • Remember margins collapse between elements
  • Use shorthand properties for cleaner code
  • Consider using relative units for responsive design
