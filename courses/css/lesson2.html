Colors in CSS üåà

Color Names

CSS has built-in color names:

  • red
  • blue
  • green
  • purple

Hex Colors

6-digit codes starting with #:

  • #FF0000 (Red)
  • #00FF00 (Green)
  • #0000FF (Blue)

RGB Colors

Using rgb() function:

  • rgb(255,0,0)
  • rgb(0,255,0)
  • rgb(0,0,255)

Typography üìù

Font Properties

  • font-family: Choose the font
  • font-size: Set text size
  • font-weight: Make text bold
  • font-style: Make text italic

Text Properties

  • text-align: Align text
  • text-decoration: Add underline
  • line-height: Space between lines
  • letter-spacing: Space between letters
