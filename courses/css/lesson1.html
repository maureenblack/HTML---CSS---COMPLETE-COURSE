CSS (Cascading Style Sheets) is what makes websites beautiful. It's like having a magic paintbrush for your HTML!

What is CSS? ü§î

CSS is a style sheet language that controls how your HTML elements look on the webpage. It handles:

  • Colors and backgrounds
  • Fonts and text styles
  • Spacing and layout
  • Animations and effects

Why Use CSS? ‚ú®

  • Makes websites visually appealing
  • Improves user experience
  • Separates design from content
  • Works on all devices

Key Points üîë

  • CSS rules have a selector and declarations
  • Selectors target HTML elements
  • Declarations go inside curly braces { }
  • Each declaration has a property and value

Three Ways to Add CSS üõ†Ô∏è

1. Inline CSS

Added directly to HTML elements:

<p style="color: blue;">Blue text</p>

2. Internal CSS

Added in the <head> section:

<style> p { color: blue; } </style>

3. External CSS

In a separate .css file:

<link rel="stylesheet" href="styles.css">

Helpful Tips! üí°

  • CSS properties are always followed by a colon :
  • Each declaration ends with a semicolon ;
  • Property names use hyphens, not spaces (font-size, not font size)
  • Colors can be names (red) or codes (#FF0000)
